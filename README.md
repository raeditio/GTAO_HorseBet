# Autobet
## Description

This script detects the `GTA_Enhanced.exe` window and automates the betting process in the "Inside Tracks" game of the GTA Online Casino content.

## Usage

1. Start GTA V and enter GTA Online.
2. Go to the Diamond Casino and sit at an "Inside Track" betting terminal.
3. Enter the Place Bet screen.
4. Run the script.

**Use at your own risk.** Automating gameplay in GTA Online may be considered a violation in Rockstar's Terms of Service and could result in an account ban or character wipe.
